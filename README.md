# Memory Game

This is a memory game using container and javascript.

# Setup

Simply open `index.html` in your web browser to start playing.

# Running the Game

1. Download or clone all project files
2. Open `index.html` in web browser
3. Start playing

# Game Logic

- To handle card flipping. When a card is clicked, it will flip to reveal the front face
- A shuffle function to randomize the card positions at the start of the game
- If two cards match, they will remain face-up. If they do not match, they will flip back after a short delay
- Continue until you've matched all 8 pairs
- Try to complete the game in as few moves as possible
- Game can be reset when you press the reset button
